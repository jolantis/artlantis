# CamelCase plugin

A plugin for [Kirby CMS](http://getkirby.com) to convert text to camelCase.

## Installation

Add the `camelcase` folder to your `/site/plugins` directory.

## How to use it?

Call from template:

```php
<?php echo camelcase(‘resultingInCamelCaseText’); ?>
```
